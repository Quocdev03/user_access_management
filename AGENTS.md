# AI Project Instructions

Before every task:

1. Read all documentation in docs/.
2. Analyze the current source code.
3. Understand existing architecture.
4. Search for similar implementations.
5. Reuse existing code whenever possible.
6. Never duplicate code.
7. Follow Clean Architecture.
8. Follow Coding Convention.
9. Do not change folder structure.
10. Do not change API standard.
11. Keep naming consistent.
12. Update documentation when necessary.

Before implementing any feature:

- Find similar handlers.
- Find similar services.
- Find similar repositories.
- Find similar DTOs.

If existing code can be extended, extend it instead of creating new code.

When unsure, ask instead of assuming.
